# СтройТехника - Строительная Техника

Русскоязычный сайт для демонстрации строительной техники и оборудования.

## Описание проекта

Современный, адаптивный веб-сайт на русском языке для компании, специализирующейся на продаже строительной техники. Сайт включает полный каталог продукции, информацию о компании, контактные данные и услуги.

## Структура сайта

### Основные страницы:
1. **index.html** - Главная страница
   - Навигация
   - Герой-секция с призывом к действию
   - Категории техники
   - Популярные товары
   - Преимущества компании
   - Футер с контактами

2. **products.html** - Каталог продукции
   - Фильтр по категориям
   - Поиск по названию
   - Карточки товаров с деталями
   - Цены и характеристики

3. **product-detail.html** - Детали товара
   - Галерея изображений
   - Технические характеристики
   - Цена и кнопки заказа
   - Похожие товары

4. **about.html** - О компании
   - История компании
   - Миссия и ценности
   - Статистика
   - Команда

5. **contact.html** - Контакты
   - Контактная информация
   - Форма обратной связи
   - Карта расположения
   - Часы работы
   - Социальные сети

6. **services.html** - Услуги
   - Основные услуги
   - Процесс работы
   - FAQ
   - Дополнительные услуги

## Установка и настройка

1. Скопируйте все файлы на хостинг или в папку проекта
2. Убедитесь, что все файлы находятся в одной директории
3. Откройте `index.html` в браузере для проверки
4. Для настройки:
   - Отредактируйте контактные данные в каждом HTML-файле
   - Замените изображения на свои
   - Обновите информацию о продукции

## Настройка для Gitee Pages

1. Создайте репозиторий на Gitee (码云)
2. Загрузите все файлы проекта
3. В настройках репозитория включите Gitee Pages
4. Выберите ветку для публикации (обычно master/main)
5. Сохраните настройки - сайт будет доступен по адресу `username.gitee.io/repository-name`

## Рекомендации по контенту

### Изображения
- Используйте качественные изображения техники
- Оптимизируйте размер изображений (рекомендуется 800-1200px по ширине)
- Сохраняйте единый стиль фотографий

### Текст
- Пишите подробные описания товаров
- Указывайте точные технические характеристики
- Добавляйте отзывы клиентов (при наличии)
- Регулярно обновляйте информацию о наличии товаров

### SEO оптимизация
- Добавьте мета-теги description и keywords
- Используйте семантические HTML-теги
- Создайте карту сайта (sitemap.xml)
- Настройте robots.txt
